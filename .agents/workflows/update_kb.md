description: Menjaga konsistensi KNOWLEDGE_BASE.md agar selalu up-to-date dengan perkembangan proyek.
---

Setiap kali terjadi perubahan signifikan pada arsitektur, penambahan fitur (modul), atau keputusan teknis penting, ikuti langkah berikut:

1.  **Identifikasi Perubahan**: Catat apa yang telah diubah atau ditambahkan (misal: modul baru, integrasi API baru).
2.  **Update KNOWLEDGE_BASE.md**: Buka file `KNOWLEDGE_BASE.md` di root proyek dan tambahkan informasi relevan pada section yang sesuai.
3.  **Tambahkan Point Baru**: Jika perubahan bersifat unik (misal: penanganan error global baru), buat section baru jika diperlukan.
4.  **Verifikasi Kelengkapan**: Pastikan dokumentasi mencerminkan status *current* dari codebase.
5.  **Informasikan User**: Beritahukan user bahwa KB telah diupdate.
